# Uptime Dashboard

Es un dashboard moderno donde se muestra el uptime en tiempo real de los servicios fundamentales para el uso diario (como Google Cloud, AWS, GitHub, Vercel, etc.). 

## Tecnologías Principales
- Next.js (App Router)
- React
- Tailwind CSS
- Lucide Icons (para iconografía)

## Cómo ejecutar en local

Primero, instala las dependencias y corre el servidor de desarrollo:

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.

## Despliegue (Deploy)

La plataforma recomendada para desplegar este proyecto es [Vercel](https://vercel.com/new), construida por los creadores de Next.js. Funciona de manera inmediata directamente desde este repositorio.
